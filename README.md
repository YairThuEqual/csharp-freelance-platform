# C--freelance-platform
A desktop freelance marketplace application built using C# (Windows Forms) and MySQL. This system allows clients and freelancers to interact through project posting, bidding/reviewing, profile management, and project tracking features.

# Overview
This project is a freelance platform designed to simulate real-world freelance marketplaces such as Fiverr or Upwork. It focuses on: 
  - Client project posting
  - Freelancer profile viewing
  - Project management
  - Reviews and ratings
  - Database-driven workflow
The application is built as a Windows desktop application using C# and integrates with a MySQL database for data persistence.

# Features
  User Management
  - User registration and login
  - Role-based system (Client / Freelancer)
  - Profile management
    
# Project Management
  - Clients can post new projects
  - View active and completed projects
  - Project status tracking

# Reviews & Ratings
  - Clients can review freelancers
  - Rating system for completed projects

# Freelancer Browsing
  - View freelancer profiles
  - Check reviews and previous work
    
# Technologies Used
  - C# (.NET Framework)
  - Windows Forms (WinForms)
  - MySQL Database
  - SQL Scripts
  - Visual Studio 2022
